IEEE Robot
==========

Arduino code relies on [SerialCommand](https://github.com/kroimon/Arduino-SerialCommand)
Python code relies on [simpleCV](http://www.simplecv.org/) and [libfreenect](https://github.com/OpenKinect/libfreenect)

Serial Commands:
* TURN [0-1023]
* SPEED [0-1023]
* SPEEDLEVEL [-1,1]
* POWER

Serial Data:
* Power [0-1023]
* "Power Toggled"
* "Speed Up"
* "Speed Down"
* "Invalid Parameter"
* "What?"
* "Ready"
* "Power Switch"
